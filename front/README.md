# Stream chat overlay - Front end

Front end for stream chat overlay built on react + typescript.


## .env
A environmental file with the following keys is required to run

```
// Required
VITE_API_URL='{{ backend api url }}'
VITE_WS_URL='{{ backend ws url }}'

// Optional
VITE_UMAMI_ID='{{ umami tracking id }}'
VITE_UMAMI_SRC_URL='{{ umami source url }}'
```
