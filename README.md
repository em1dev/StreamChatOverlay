## ⭐ Features

* 100% Free and open source
* Text to speech support
* BetterTTV, FFZ and 7TV integration
* Font and size customization
* Pronoun support via [pronouns.alejo.io](https://pr.alejo.io)

# 💽 Backend

Built with node express and typescript. Requires a connection to [EmyDev AuthService](https://github.com/em1dev/AuthService)

Documentation - [Readme](back/README.md)
